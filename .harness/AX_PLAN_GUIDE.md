# AX Plan Guide

1. 사용자 요청이 기존 plan.md 범위 내인지 확인합니다.
2. 새 범위라면 plan.md(milestone)와 progress-tracker.md(T-id) 양쪽에 추가합니다.
3. 사용자가 범위를 드랍하면 삭제 대신 dropped 표기를 추가합니다.
4. 신규 프로젝트는 코딩 전에 plan.md/project-brief.md를 채웁니다.
